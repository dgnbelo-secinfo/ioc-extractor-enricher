# IOC Extractor & Enricher

Python tool developed to extract Indicators of Compromise (IOCs) from raw text data.

## Features

* IP extraction
* URL extraction
* Domain extraction
* MD5 detection
* SHA256 detection

## Technologies

* Python 3
* Regex

## Usage

```bash
python ioc_extractor.py
```

## IOC Types Supported

* IPv4
* URLs
* Domains
* MD5
* SHA256

## Objective

This project was created for cybersecurity portfolio purposes focused on SOC operations, Threat Hunting, and Incident Response workflows.
